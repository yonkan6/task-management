Пітч: Веб-застосунок, призначений для керування (створення, редагування, виконання) тими чи іншими завданнями. Він допомагає учням, студентам, та командам у зручному форматі контролювати наявні дедлайни та підвищувати показники особистої ефективності шляхом впровадження автоматизованих засобів планування. Зі сторони керівної частини (викладачі, керівна команда), це зручний спосіб донести до виконавців актуальні завдання, контролювати їхній статус та своєчасність виконання. Тобто систеа довзволяє спостерігат изручний та прозорий процес керування завданнями для всіх учасників - від виконавців до керівників -   забезпечуючи організованість, контроль дедлайнів і підвищення ефективності роботи.
