Пітч: Веб-застосунок, призначений для керування (створення, редагування, виконання) завданнями. Інструмент допомагає учням, студентам, та командам у зручному форматі контролювати наявні дедлайни та підвищувати показники особистої ефективності шляхом впровадження автоматизованих засобів планування. Зі сторони керівної частини (викладачі, керівна команда) це зручний спосіб донести до виконавців актуальні завдання, контролювати їхній статус та своєчасність виконання. Тобто система довзволяє спостерігат изручний та прозорий процес керування завданнями для всіх учасників - від виконавців до керівників -   забезпечуючи організованість, контроль дедлайнів і підвищення ефективності роботи.
Проблема: Наявність великого обсягу роботи, що важко контролювати, внаслідок чого утворюються затримки в роботі та прострочені дедлайни. Відсутність інструмента для зручного планування та компонування роботи, що передбачає навчальний або робочий процес.
Ціль першої ітерації: Мінімальна версія застосунку, що передбачає наступні можливості: прикріплення хоча б одного або двох завдань; можливість перегляду завдання у списку завдань та детальної інформації завдання;
Ядро цінності: Надання користувачу можливість легко та зручно переглянути свої завдання і контролювати їх виконання.
